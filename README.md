# Data Engineering - Pragmatic Notes

> *"Software engineering is what happens to programming when you add time and other programmers."* — **Titus Winters (Google)**

Este repositorio no es solo una colección de código; es un **laboratorio de decisiones técnicas**. Aquí documento la aplicación de principios de Ingeniería de Software de alto nivel en el ecosistema de **Data Engineering** y **Sistemas Distribuidos**.

## Objetivo
Cerrar la brecha entre los conceptos académicos/teóricos de libros como *A Philosophy of Software Design* y *The Pragmatic Programmer* con la realidad de construir pipelines de datos robustos, escalables y mantenibles, con un enfoque especial en sistemas de **NLP, ML, IA, etc**.

## Fuentes de Pensamiento
- **Software Engineering at Google** (Winters, Manshreck, Wright)
- **A Philosophy of Software Design** (John Ousterhout)
- **The Pragmatic Programmer** (Hunt & Thomas)
- **System Design Interview** (Alex Xu)
